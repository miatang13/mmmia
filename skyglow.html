Skills: Python / MATLAB / Computational Design

A constellation recognition app written in Python.

Big Ideas /

I've always been fascinated by the stars, the outer space.
Hence, I decided to create a constellation recognition software aimed to let everyone see the hidden constellation patterns in the ordinary environments, thus finding the hidden stars around them.

Design Process 1 / Personas

I created personas to have a better idea of my target audience and how I can design a program to inspire their curiosity and appreciation of the stars. As a conclusion, I want to focus on educating basic astronomical knowledge with a heavy emphasis on motivating appreciation of stars.

Design Process 2 / Story-Boarding


Before diving into the technical implementation part, I visualized what I want to achieve using illustrations and Photoshop.

Users take a photo in their everyday environment to identify the constellation (star) patterns they see, then the program will evaluate the given coordinates.

Design Process 3 / User Flow

I wanted a rather simple user flow that is short but still effective. The app is targeted at learning and inspiring, hence it also motivates frequent usage by having the "constellation collection" storage feature.

Algorithm /
How do I recognize constellation patterns without AI or ML?

My first few ideas were all related to either Artificial Intelligence or Machine Learning, but I wanted to challenge myself and come up with an efficient Algorithm that recognizes constellation patterns by basic arithmetics.

Step 1 / User Input

Users would upload a single image to identify the constellation points they see. The input data will then be stored to be processed.

Step 2 & 3 / Determine Anchors

two input coordinates are chosen to be anchors if they are the furthest points apart in the set of given user input. Then, the entire set of points will be translated horizontally and vertically with Anchor 1, which will be translated to the origin, (0,0). 


            import math
            import numpy as np
            import matplotlib.pyplot as plt
            import copy

            def distance(x1, y1, x2, y2):
                return math.sqrt( (x2 - x1)**2 + (y2 - y1)**2 )
            
            def coordToPolar(x, y):
                r = distance(x, y, 0, 0)
                theta = np.arctan2([y], [x])[0]
                return r, theta
            
            def polarToCoord(r, theta):
                x = r * np.cos(theta)
                y = r * np.sin(theta)
                return x, y
            

            #Input: data as a 1D list of coordinates (tuples)
            #Return: the most distant two points, or None otherwise
            def comparePoints(points):
                if points == [] or len(points) == 1:
                    return None  
                #the farthest distance between one point and rest
                longestDistance = -1 
                farthestPoint1 = ()
                farthestPoint2 = ()
                for i in range(len(points)-1): 
                    x1 = points[i][0]
                    y1 = points[i][1]   
                    for j in range(i + 1, len(points)):
                        x2 = points[j][0]
                        y2 = points[j][1]          
                        curDistance = distance(x1, y1, x2, y2)
                        if curDistance > longestDistance:
                            longestDistance = curDistance
                            farthestPoint1 = (x1, y1)
                            farthestPoint2 = (x2, y2)
                return (farthestPoint1, farthestPoint2)

Step 4 & 5 / Rotate Points

The delta theta θ is calculated by the θ formed by rotating Anchor 2 onto the horizontal x-axis. Then, the rotation with the same θ will be applied to each and every coordinate. 


        #This function outputs the shifted points with one anchor on (0, 0)
        #input: anchor(tuple), the other farthest point, rest of the points
        #output: anchor, farCord, remainingPoints
        def centerPoints(anchor, farCord, remainingPoints):
            offsetX = anchor[0] - 0
            offsetY = anchor[1] - 0
            shiftedAnchor = (anchor[0] - offsetX, anchor[1] - offsetY)
            shiftedFarCord = (farCord[0] - offsetX, farCord[1] - offsetY)
            newRemainPoints = []
            for i in range(len(remainingPoints)):
                theX = remainingPoints[i][0]
                theY = remainingPoints[i][1]
                newRemainPoints.append((theX - offsetX, theY - offsetY))
            return (shiftedAnchor, shiftedFarCord, newRemainPoints)

        def rotateRemain(newRemainPoints, shiftTheta):
            rotatedList = []
            for i in range(len(newRemainPoints)):
                r, theta = coordToPolar(newRemainPoints[i][0], newRemainPoints[i][1])
                newTheta = theta - shiftTheta
                newX, newY = polarToCoord(r, newTheta)
                rotatedList.append((newX, newY))
            return rotatedList
                

        def rotateFarCord(shiftedFarCord, shiftTheta):
            r, theta = coordToPolar(shiftedFarCord[0], shiftedFarCord[1])
            newTheta = theta - shiftTheta
            newX, newY = polarToCoord(r, newTheta)
            rotatedFarCord = (newX, newY)
        return rotatedFarCord
        
        #Input: Anchored coordinates 
        #Return: Rotate coordinates
        def rotateFlat(shiftedAnchor, shiftedFarCord, newRemainPoints):
            farX = shiftedFarCord[0]
            farY = shiftedFarCord[1]
            shiftTheta = math.atan(farY / farX)
            rotatedFarCord = rotateFarCord(shiftedFarCord, shiftTheta)
            rotatedRemain = rotateRemain(newRemainPoints, shiftTheta)
        return (shiftedAnchor, rotatedFarCord, rotatedRemain)

        #Return: scaled user input coordinates
        def scale(starFarthest, shiftedFarCord, newRemainPoints):
        ratio  = starFarthest / shiftedFarCord[0]
        scaledPoints = []
        finalFarCord = starFarthest
        for i in range(len(newRemainPoints)): #all the other points
            newX = newRemainPoints[i][0] * ratio
            newY = newRemainPoints[i][1] * ratio
            scaledPoints.append((newX, newY))
        return finalFarCord, scaledPoints

Step 6 / Evaluate Input

Scores will be calculated comparing the given input with the set of correct constellation coordinates for the highest scores as potential solutions.


            def totalDifference (drawnPoints, constellationPoints):
            differences = abs(len(drawnPoints) - len(constellationPoints)) * 5
            starPoints = copy.deepcopy(constellationPoints)
