SkyGlow is a constellation recognition app written in Python. This project inspired me to take more computer science classes and further explore into the field of Computational Design.

Programming | CMU School of Computer Science Summer 2019

Duration: 1 Week
Skills: Python / MATLAB / Computational Design / Creative Programming

Big Ideas /

I've always been fascinated by the stars, the outer space.

Hence, I decided to create a constellation recognition software aimed to let everyone see the hidden constellation patterns in the ordinary environments, thus finding the hidden stars around them.

Design Process 1 / Personas

I created personas to have a better idea of my target audience and how I can design a program to inspire their curiosity and appreciation of the stars.

Design Process 2 / User Flow

Due to my 1 week time constraint, I had to settle with a rather simple user flow that is short but still effective.

Part A /
Constellation Recognition

Upload image to identify the constellation points you see. The input data will then be processed and stored.

The input coordinates will be translated onto the x-axis for easier comparison with the correct constellation coordinates. After rotation, the furthest two points will be calculated and chosen as the anchors to compare the remaining points. Scores will be calculated comparing the given input with the set of correct constellation coordinates for the highest scores as potential solutions.

The constellation coordinates were manually entered and recorded.

Part B /
Computational Graphics

Interactive bouncy ball built using geometry, vectors, and simple kinematics.

The hardest part of this was to consider some special cases that will only occur in a computer simulated world where the time is not continuous but discrete. This led me to realize how sometimes some approximation should be creatively used in order to avoid solving complicated equations and to improve performance.
